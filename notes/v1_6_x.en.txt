v1.6.0
New And Updated Features
<ul>
<li>Create directory for each tab when saving temporary files.</li>
<li>Also, tab's temporary files are saved by each host.</li>
<li>Add support for `data URI`s.</li>
<li>Add `Enable Only When Editing Text` option.</li>
</ul>

Code Changes
<ul>
<li>Partially `class` based.</li>
<li>Added `close` event listener to the tab and removes temporary stored data when the tab is closed.</li>
<li>Removed removeListeners in `onUnload` function.</li>
<li>Checked JavaScript code style with eslint.</li>
</ul>
