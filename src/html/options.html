withExEditor Options

Toolbar Button Icon

Choose the icon.

NOTE: "Auto" only applies to Gecko browsers, and you need to set "svg.context-properties.content.enabled" to "true" from "about:config".

Host

withExEditorHost

Notifies when the latest version of the host is available.

If you are using a binary host, get the latest version from the link. If you are using the npm package, update with the command "npm i -g withexeditorhost".

Host Connection Unknown

If the status is "Disconnected" or "Unknown" instead of "Connected", reload withExEditor.

Host Compatibility Unknown

If the status is "Incompatible" or "Unknown" instead of "Compatible", update the host.

Editor State Unknown

If the status is "Not Executable" or "Unknown" instead of "Executable", run the setup script on the host.

Enter the label of the editor.

If the input field is disabled, run the setup script on the host.

Enable this option if you want to be notified when a host fails.

Edit Text

Enable this option if you want to use this extension only when the content is editable.

Adds a context menu item for selecting a temporary file extension.

Enable this option if you want to automatically synchronize edited content.

Note that this option is disabled during private browsing. Also note that if you leave your browser running for a long time or register many URLs, it may affect the performance of the PC.

Reload withExEditor

Try when withExEditor stops working, for example when updating Node.js or updating withExEditorHost.
